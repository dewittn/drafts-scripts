# Archived Tests

This directory contains tests that are no longer actively maintained but are kept for historical reference.

## Singleton/Memory Tests
- `singleton-memory-test.js`
- `singleton-memory-test-simple.js`
- `singleton-test-inline.js`
- `singleton-validation-test.js`

These were experimental tests exploring different singleton patterns and memory management approaches. They were useful during development but are not part of the regular test suite.

## One-Time Validation Tests
- `manager-extraction-test.js`

This was a validation test created during the manager extraction refactoring to verify that the ContentPipeline API remained unchanged. It served its purpose and is archived for historical reference.

## Why Archived?

These tests are archived rather than deleted because:
1. They document implementation decisions and experiments
2. They may be useful for future reference
3. They provide historical context for architectural changes

They are not part of the regular test suite and should not be run as part of standard testing workflows.
